# Setup

-   Clone repo

```console
git clone https://github.com/michaelbui99/terminal-configs.git
```

-   Navigate to /terminal-configs

```console
cd .\terminal-configs
```

-   Run setup_terminal_windows

```console
.\setup_terminal_windows.ps1
```

## .ideavimrc

```vim
set number
set relativenumber
set ideajoin
set ignorecase
set smartcase
set ignorecase
set smartcase
set tabstop=4
set softtabstop=4
set expandtab
```
